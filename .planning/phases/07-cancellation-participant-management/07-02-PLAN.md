Implement 4 export methods. All exports filter `!r.IsCancelled` (only active registrations per locked decision).

**Method 1: ExportParticipantListAsync(int eventId) → byte[]** (PART-02)
- Query: Registrations where EventId=eventId and !IsCancelled
- Include EventCompany for company name
- Columns: Vorname, Nachname, E-Mail, Firma, Typ (Makler/Gast/Firma), Anmeldedatum
- Company name logic: `r.EventCompany?.CompanyName ?? r.Company ?? "N/A"`
- Type mapping: RegistrationType.Makler→"Makler", Guest→"Gast", Company→"Firma"
- Date format: CET conversion using TimeZoneHelper.ConvertUtcToCet, formatted "dd.MM.yyyy HH:mm"
- Sheet name: "Teilnehmerliste"
- Use `ws.Cell("A1").InsertTable(data)` then `ws.Columns().AdjustToContents()`
- Return `byte[]` via MemoryStream pattern (use `using` for both XLWorkbook and MemoryStream)

**Method 2: ExportContactDataAsync(int eventId) → byte[]** (PART-03)
- Query: Same filter as above
- Columns: Vorname, Nachname, E-Mail, Telefon, Firma
- Phone: `r.Phone ?? "N/A"`
- Sheet name: "Kontaktdaten"

**Method 3: ExportNonParticipantsAsync(int eventId) → byte[]** (PART-04)
- This calculates the delta: invited company members minus registered members
- Query EventCompanies where EventId=eventId and Status is Sent or Booked (companies that received invitations)
- Include Registrations (filtered by !IsCancelled) for each company
- For each company: `notRegistered = Math.Max(0, (MaxParticipants ?? 0) - registrations.Count)`
- Skip companies with MaxParticipants == null (per research recommendation: no delta calculable)
- Skip companies with notRegistered == 0
- Columns: Firma, Kontakt-E-Mail, Kontakt-Telefon, Eingeladene, Registrierte, Nicht registriert
- Sheet name: "Nicht-Teilnehmer"

**Method 4: ExportCompanyListAsync(int eventId) → byte[]** (PART-05)
- Query EventCompanies where EventId=eventId
- Include Registrations for participant count
- Columns: Firma, Kontakt-E-Mail, Kontakt-Telefon, Status, Teilnehmer (count of !IsCancelled registrations), Einladung gesendet (date)
- Status mapping: InvitationStatus to German string (Draft→"Entwurf", Sent→"Gesendet", Booked→"Gebucht", Cancelled→"Storniert", NonParticipation→"Nicht-Teilnahme")
- Date format: CET conversion, "dd.MM.yyyy" or "N/A" if null
- Sheet name: "Firmenliste"
