<feature>
  <name>CompanyInvitationService — Invitation CRUD, pricing, and email triggering</name>
  <files>EventCenter.Web/Services/CompanyInvitationService.cs, EventCenter.Tests/Services/CompanyInvitationServiceTests.cs, EventCenter.Tests/Validators/CompanyInvitationValidatorTests.cs</files>
  <behavior>
    **GUID Generation:**
    - GenerateSecureInvitationCode() -> 32 hex char string (no dashes)
    - Uses RandomNumberGenerator.Fill(byte[16]) with RFC 4122 v4 formatting
    - Each call returns unique value
    - Format: valid GUID parseable string

    **Pricing Calculation:**
    - CalculateCustomPrice(basePrice=100, percentageDiscount=10, manualOverride=null) -> 90.00
    - CalculateCustomPrice(basePrice=100, percentageDiscount=10, manualOverride=85) -> 85.00 (override takes precedence)
    - CalculateCustomPrice(basePrice=100, percentageDiscount=null, manualOverride=null) -> 100.00 (no discount)
    - CalculateCustomPrice(basePrice=99.99, percentageDiscount=33.33, manualOverride=null) -> 66.66 (rounded AwayFromZero)
    - Always uses decimal math, Math.Round(value, 2, MidpointRounding.AwayFromZero)

    **Create Invitation (single):**
    - Valid input -> (true, invitationId, null), creates EventCompany + EventCompanyAgendaItemPrices
    - Event not found -> (false, null, "Veranstaltung nicht gefunden.")
    - Duplicate email for same event -> (false, null, "Diese Firma wurde bereits eingeladen.")
    - SendImmediately=false -> Status=Draft, InvitationSentUtc=null
    - SendImmediately=true -> Status=Sent, InvitationSentUtc=now, email triggered
    - Per-item pricing stored correctly: CustomPrice set to calculated value only if different from base price

    **Create Batch Invitations:**
    - CreateBatchInvitationsAsync(eventId, List<CompanyInvitationFormModel>) -> creates multiple invitations
    - Skips duplicates (same email), reports which succeeded and which failed
    - All use standard pricing (no per-item overrides in batch mode)

    **Send/Resend Invitation:**
    - SendInvitationAsync(invitationId) for Draft -> Sent transition, sets InvitationSentUtc, triggers email
    - ResendInvitationAsync(invitationId) for Sent status -> updates InvitationSentUtc, triggers email again
    - Invalid status -> returns error message

    **Update Invitation:**
    - UpdateInvitationAsync(invitationId, formModel) -> updates pricing, personal message, contact details
    - Works in ALL statuses (pricing always editable per user decision)
    - Replaces AgendaItemPrices collection (delete old, create new)

    **Delete Invitation:**
    - DeleteInvitationAsync(invitationId) -> soft considerations:
      - Draft: hard delete allowed
      - Sent: hard delete allowed
      - Booked: NOT allowed -> (false, "Diese Einladung kann nicht gelöscht werden, da bereits eine Buchung vorliegt.")
      - Cancelled: hard delete allowed

    **Get Invitations:**
    - GetInvitationsForEventAsync(eventId) -> list with navigation properties loaded
    - GetInvitationByIdAsync(invitationId) -> single with AgendaItemPrices.AgendaItem loaded

    **Status Query:**
    - GetInvitationStatusSummaryAsync(eventId) -> counts per status for overview

    **Validator Tests:**
    - Empty CompanyName -> validation fails with "Firmenname ist erforderlich"
    - Invalid email -> validation fails with "Ungültige E-Mail-Adresse"
    - PercentageDiscount = 101 -> validation fails
    - PercentageDiscount = -1 -> validation fails
    - ManualOverride = -5 -> validation fails with "Preis darf nicht negativ sein"
    - Valid model -> validation passes
  </behavior>
  <implementation>
    **CompanyInvitationService class:**
    - Constructor: inject EventCenterDbContext, IEmailSender, ILogger<CompanyInvitationService>, IConfiguration (for building invitation URLs)
    - Use `Database.BeginTransactionAsync()` for create operations (same pattern as RegistrationService)
    - Fire-and-forget email: `Task.Run(async () => { try { ... } catch { _logger.LogError(...); } })` after transaction commit
    - Build invitation link: `$"{baseUrl}/company/booking/{invitationCode}"` where baseUrl from IConfiguration
    - GenerateSecureInvitationCode: RandomNumberGenerator.Fill(byte[16]), set version 4 bits, return Guid.ToString("N")
    - CalculateCustomPrice: static method, decimal math, MidpointRounding.AwayFromZero

    - Mock ILogger and IConfiguration
