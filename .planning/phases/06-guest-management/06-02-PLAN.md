Modify OnInitializedAsync:
- After checking `isUserRegistered`, if true: find the broker's Registration from `evt.Registrations` matching `userEmail` (non-cancelled, RegistrationType.Makler), store its Id in `userBrokerRegistrationId`
- Call `await LoadGuestDataAsync()` to load guest state

New methods:

**LoadGuestDataAsync():**
```csharp
private async Task LoadGuestDataAsync()
{
    if (userBrokerRegistrationId.HasValue)
    {
        currentGuestCount = await RegistrationService.GetGuestCountAsync(userBrokerRegistrationId.Value);
        userGuestRegistrations = await RegistrationService.GetGuestRegistrationsAsync(userBrokerRegistrationId.Value);
        totalGuestCost = userGuestRegistrations.Sum(g =>
            g.RegistrationAgendaItems.Sum(rai => rai.AgendaItem.CostForGuest));
    }
}
```

**HandleGuestRegistrationAsync():**
- Set isSubmittingGuest = true, clear messages
- Call `RegistrationService.RegisterGuestAsync(userBrokerRegistrationId.Value, guestFormModel.Salutation!, guestFormModel.FirstName, guestFormModel.LastName, guestFormModel.Email, guestFormModel.RelationshipType!, guestFormModel.SelectedAgendaItemIds)`
- On success: set guestSuccessMessage = "Begleitperson erfolgreich angemeldet.", collapse form (showGuestForm = false), reset guestFormModel, call LoadGuestDataAsync()
- On failure: set guestErrorMessage = result.ErrorMessage
- Set isSubmittingGuest = false

**CancelGuestForm():**
- showGuestForm = false
- Reset guestFormModel = new()
- Clear messages
- Reset selectedGuestAgendaItemIds

**ToggleGuestAgendaItem(int agendaItemId):**
- Toggle item in selectedGuestAgendaItemIds
- Sync to guestFormModel.SelectedAgendaItemIds
- Recalculate live cost: sum of CostForGuest for selected items

**CalculateSelectedGuestCost():**
- Return sum of CostForGuest for items in selectedGuestAgendaItemIds from evt.AgendaItems

**5. Agenda item checkbox implementation** — Use individual checkboxes with @onchange handler (not InputCheckbox with bind, which doesn't work well for collections). Pattern:

```razor
@foreach (var item in guestAgendaItems)
{
    var itemId = item.Id;
    var isSelected = selectedGuestAgendaItemIds.Contains(itemId);
    var isMandatory = item.IsMandatory;

    <div class="form-check">
        <input type="checkbox" class="form-check-input"
               checked="@(isSelected || isMandatory)"
               disabled="@isMandatory"
               @onchange="() => ToggleGuestAgendaItem(itemId)" />
        <label class="form-check-label">
            @item.Title
            @if (item.CostForGuest > 0)
            {
                <span class="badge bg-primary ms-2">@item.CostForGuest.ToString("N2") EUR</span>
            }
            else
            {
                <span class="badge bg-success ms-2">Kostenfrei</span>
            }
        </label>
    </div>
}
```
