**2. Extend RegistrationService.cs** — Add three new methods following the existing RegisterMaklerAsync() pattern:

**a) RegisterGuestAsync()** — Signature: `public async Task<(bool Success, int? GuestRegistrationId, string? ErrorMessage)> RegisterGuestAsync(int brokerRegistrationId, string salutation, string firstName, string lastName, string email, string relationshipType, List<int> selectedAgendaItemIds)`

Logic (within transaction):
1. Load broker registration with `.Include(r => r.Event).ThenInclude(e => e.AgendaItems).Include(r => r.Event.Registrations)` — verify it exists and is not cancelled
2. Verify `brokerRegistration.RegistrationType == RegistrationType.Makler` (only brokers can register guests) — error: "Nur Makler konnen Begleitpersonen anmelden."
3. Validate event state via `evt.GetCurrentState() == EventState.Public` — error: "Anmeldung nicht moglich - Frist abgelaufen."
4. Check companion limit: count existing guests via `_context.Registrations.CountAsync(r => r.ParentRegistrationId == brokerRegistrationId && !r.IsCancelled)`, compare against `evt.MaxCompanions` — error: "Maximale Anzahl Begleitpersonen erreicht ({evt.MaxCompanions})."
5. Validate selected agenda items: only items where `GuestsCanParticipate == true` are valid — error: "Ungultige Agendapunkt-Auswahl fur Begleitperson."
6. Create Registration with `RegistrationType = RegistrationType.Guest`, `ParentRegistrationId = brokerRegistrationId`, Salutation, RelationshipType, IsConfirmed = true
7. Create RegistrationAgendaItem entries for selected items
8. SaveChanges + CommitTransaction
9. Fire-and-forget email: `SendGuestRegistrationConfirmationAsync(guestRegistration, brokerRegistration)` using Task.Run with try-catch logging (same pattern as RegisterMaklerAsync)
10. Return success with guestRegistration.Id

Error handling: wrap in try-catch, rollback on failure, log errors, return generic error message.

**b) GetGuestCountAsync()** — `public async Task<int> GetGuestCountAsync(int brokerRegistrationId)` — Returns count of non-cancelled guests for a broker registration.

**c) GetGuestRegistrationsAsync()** — `public async Task<List<Registration>> GetGuestRegistrationsAsync(int brokerRegistrationId)` — Returns guest registrations with `.Include(r => r.RegistrationAgendaItems).ThenInclude(rai => rai.AgendaItem)`, filtered by `ParentRegistrationId == brokerRegistrationId && !r.IsCancelled`, ordered by `RegistrationDateUtc`.
