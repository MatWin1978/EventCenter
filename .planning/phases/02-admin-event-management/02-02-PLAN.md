  <action>
    **TDD RED — Write tests first (see Task 2). If running in Wave 1 parallel with Plan 01, ensure Plan 01 entity changes exist first. If not, create a minimal version of the service that works with current entities and extend after Plan 01 completes.**

    **Create EventService.cs** in `EventCenter.Web/Services/`:

    ```csharp
    public class EventService
    {
        private readonly EventCenterDbContext _context;
        private readonly IWebHostEnvironment _environment;

        public EventService(EventCenterDbContext context, IWebHostEnvironment environment) { ... }
    }
    ```

    **Methods to implement:**

    1. `async Task<Event> CreateEventAsync(Event evt)` — Add event to context, save, return with Id.

    2. `async Task<Event?> GetEventByIdAsync(int id)` — Load event with Include(AgendaItems), Include(EventOptions), Include(Registrations). Return null if not found.

    3. `async Task<List<Event>> GetEventsAsync(bool includePast, string? sortColumn, bool ascending, int page, int pageSize)` — Query events with optional filtering (if !includePast, filter EndDateUtc >= DateTime.UtcNow), sorting by the specified column (Title, StartDateUtc, Location; default: StartDateUtc descending), and pagination (Skip/Take). Include Registrations for count display.

    4. `async Task<int> GetEventCountAsync(bool includePast)` — Count for pagination.

    5. `async Task UpdateEventAsync(Event evt)` — Update event entity. Do NOT update AgendaItems or EventOptions here — those are managed separately via inline editing on the form.

    6. `async Task<bool> PublishEventAsync(int eventId)` — Set IsPublished = true. Return true on success.

    7. `async Task<(bool Success, string? ErrorMessage)> UnpublishEventAsync(int eventId)` — Check if event has any Registrations. If yes, return (false, "Veranstaltung kann nicht zurückgezogen werden, da bereits Anmeldungen existieren."). If no registrations, set IsPublished = false.

    8. `async Task<bool> DeleteEventAsync(int eventId)` — Check for registrations. If registrations exist, return false. Otherwise delete (cascade handles AgendaItems and Options).

    9. `async Task<Event> DuplicateEventAsync(int sourceEventId)` — Load source event with AgendaItems and EventOptions. Create new Event copying all fields except: Id=0, IsPublished=false, Title = "{source.Title} (Kopie)", dates shifted by +1 month. Copy all AgendaItems (new Ids, preserve relative time offsets from event start). Copy all EventOptions (new Ids). Do NOT copy Registrations or Companies. Save and return new event.

    10. `async Task<(bool Success, string? ErrorMessage)> DeleteEventOptionAsync(int optionId)` — Load EventOption with Include(Registrations). If Registrations.Any(), return (false, "Diese Zusatzoption kann nicht gelöscht werden, da bereits Buchungen existieren."). Otherwise remove and save.

    11. `async Task<string> SaveDocumentAsync(int eventId, string fileName, Stream content)` — Save file to `wwwroot/uploads/events/{eventId}/` directory. Generate unique filename: `{Guid.NewGuid()}_{sanitizedFileName}`. Return relative path `/uploads/events/{eventId}/{uniqueFileName}`. Create directory if not exists. Use IWebHostEnvironment.WebRootPath for base path.

    12. `async Task DeleteDocumentAsync(string relativePath)` — Delete file from filesystem. Use IWebHostEnvironment.WebRootPath to resolve full path. Validate path starts with `/uploads/events/` to prevent path traversal.

    **Register in DI** — Add `builder.Services.AddScoped<EventService>();` to Program.cs.
